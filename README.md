## 简介

`vue-el-admin-tpl` 是一个基于[vue-element-admin](http://panjiachen.github.io/vue-element-admin)后台集成解决方案，具体文档可以直接参考[vue-element-admin](http://panjiachen.github.io/vue-element-admin)，本项目只是介绍如何快速的使用`vue-element-admin`来完成一个后台管理的项目。
